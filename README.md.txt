# 🃏 Super Trunfo de Cidades  

Projeto em **C** que simula o cadastro de cartas no estilo *Super Trunfo*, utilizando cidades brasileiras como tema.  
Cada carta possui informações detalhadas e cálculos automáticos de propriedades estratégicas, como **densidade populacional** e **PIB per capita**.  

---

## 🚀 Funcionalidades  

- 4 cartas já cadastradas automaticamente: Salvador (BA), São Paulo (SP), Rio de Janeiro (RJ) e Belo Horizonte (MG).  
- Exibição das cartas cadastradas no terminal.  
- Cálculo automático de:  
  - **Densidade populacional**  
  - **PIB per capita**  
- Estrutura organizada em **struct** + funções.
